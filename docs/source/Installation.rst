Installation
=============

1. Create an environment and activate it

- For Windows: download yaml `here <https://github.com/hmandela/WASS2S/blob/main/WAS_S2S_windows.yml>`_ and run

.. code-block:: bash

   conda env create -f WAS_S2S_windows.yml
   conda activate WASS2S

- For Linux: download yaml `here <https://github.com/hmandela/WASS2S/blob/main/WAS_S2S_linux.yml>`_ and run

.. code-block:: bash

   conda env create -f WAS_S2S_linux.yml
   conda activate WASS2S

2. Install the wass2s package

.. code-block:: bash

   pip install wass2s   

3. Upgrade the wass2s package

.. code-block:: bash

   pip install --upgrade wass2s
