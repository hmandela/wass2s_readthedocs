User Guide
==========

This guide covers every step of the wass2s workflow in detail.
For a self-contained five-minute introduction, start with the
:doc:`Quick Start <quickstart>` page.

Comprehensive training materials — including slides, recorded sessions, and
extended exercises — are available at the
`WAS-S2S Training Portal <https://hmandela.github.io/WAS_S2S_Training/>`_.

Example notebooks
-----------------

Interactive Jupyter notebooks are available at
`github.com/hmandela/WASS2S_notebooks <https://github.com/hmandela/WASS2S_notebooks>`_.

Clone them with:

.. code-block:: bash

   git clone https://github.com/hmandela/WASS2S_notebooks.git

Or download a zip archive:

.. code-block:: bash

   wget https://github.com/hmandela/WASS2S_notebooks/archive/refs/heads/main.zip \
        -O WASS2S_notebooks.zip
