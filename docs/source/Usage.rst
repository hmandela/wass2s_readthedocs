Usage
=====
Comprehensive usage guidelines, including data download, processing, models description, configuration and execution, cross-validation, and verification, are available in the `Training Documentation <https://hmandela.github.io/WAS_S2S_Training/>`_. 
But for a quick start, use the `example notebooks <https://github.com/hmandela/WASS2S_notebooks>`_.


Download example notebooks:

.. code-block:: bash

   git clone https://github.com/hmandela/WASS2S_notebooks.git

or download the zip file:

.. code-block:: bash

   wget https://github.com/hmandela/WASS2S_notebooks/archive/refs/heads/main.zip -O WASS2S_notebooks.zip
